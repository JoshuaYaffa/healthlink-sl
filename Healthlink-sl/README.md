# 🏥 HealthLink SL - Sierra Leone Healthcare Directory

## 🌍 Digital Public Good for Healthcare Access in Sierra Leone

HealthLink SL is a **complete, production-ready healthcare directory** that connects citizens of Sierra Leone to medical facilities, emergency services, and health information across all **16 districts**.

## ✨ Features

### Core Functionality
- ✅ **500+ Clinics** across all 16 districts of Sierra Leone
- ✅ **Online + Offline Mode** (PWA - works without internet)
- ✅ **Location-Based Search** - Find nearest clinics using geolocation
- ✅ **Emergency SOS Button** - 24/7 emergency assistance
- ✅ **Contact Form** - Real email integration (Formspree)
- ✅ **Admin Panel** - Complete clinic management (CRUD operations)

### User Features
- ✅ **Dark Mode** - Switch between light/dark themes
- ✅ **Multi-Language** - Google Translate support
- ✅ **WhatsApp Integration** - Direct chat with clinics
- ✅ **Click-to-Call** - One-tap phone dialing
- ✅ **Clinic Ratings** - User ratings for each facility
- ✅ **Verified Badges** - Government-verified clinics

### Technical Features
- ✅ **Responsive Design** - Desktop, Tablet, Mobile
- ✅ **PWA Installable** - Install as native app
- ✅ **Service Worker** - Offline caching
- ✅ **LocalStorage** - Persistent data storage
- ✅ **Export/Import** - JSON and CSV support
